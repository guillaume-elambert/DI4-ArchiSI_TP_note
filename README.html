2. API Documentation

2.1. List products


Endpoint : /products
Method : GET


The following parameters can be used individually or combined.

Query parameters
order string

Sorts products on price.

Can be one of : asc or desc
Default : asc
category string
Filters products on their category label.
currency string

Returns the products with a new field : priceInCurrency containing the price of the product converted to the specified currency.

Can be any international currency abbreviations   (i.e. "usd", "gbp",...)
Non case sensitive.



2.2. Get a product


Endpoint : /products/{product_id}

2.3. Create a product


Endpoint : /products
Method : POST


Body parameters
label int
The label of the product.
category string
The product category label.
stock int
The product stock quantity.
priceInEuro or priceInCurrency double
The price of the product in :
  • Euro if priceInEuro specified
  • currency if both currency and priceInCurrency parameters are specified.

Query parameters
currency string

Update the price of the product using a conversion from the specified currency to Euro. See priceInCurrency.

Can be any international currency abbreviations   (i.e. "usd", "gbp",...)
Non case sensitive.
